Instructions

Listed below are important terms and concepts in JavaSript and jQuery. Click on each term for various elements that contain definitions of common JS and jQuery terms.

  • JavaScript

    • An object oriented computer programming language commonly used to create interactive features on web browsers.

  • Operators

    • An operator performs some operation on single or multiple operands. For example the + sign is an operator.

  • Variables

    • Variables are declared using the let or const keyword. Let allows a variable to change over time, where a variable defined with const is fixed.

  • Variables Naming Convention

    • Variables are named using lower case camel. This means the first letter of the first word is lower case while the first letter of any following words are upper case.

  • Functions

    • A function is declared using the function keyword. Functions may be hoisted or nonhoisted.

  • Methods

    • Methods are actions that may be performed on objects such as functions.

  • Arguments

    • Arguments are defined parameters that are passed into function parameters.

  • Parameters

    • placeholder variables that a function takes as input. By convention, functions should take a maximum of two or three parameters.

  • Returns

    • The return statement stops the execution of a function and returns a value from that function.
